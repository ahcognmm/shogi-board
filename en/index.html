Usage

ブラウザで動作

PC と最新版 GoogleChrome だけあれば動作する、とても軽量な Web アプリケーションです。macOS を完全にサポートし、将棋ソフトを扱う機能も開発中です。

検討・棋譜並べに最適

ルール通りに駒を動かすことができます。王手放置・打ち歩詰めも防止します。分岐の作成も可能。棋譜の保存・取り込み機能などは今年中にリリースします。

Getting Started

ダウンロード不要で試すには Playground から
Git・Node.js を扱える方は以下のように始めることができます

$ git clone https://github.com/murosan/shogi-board.git
$ cd shogi-board
$ yarn
$ yarn start
