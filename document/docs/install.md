インストールせずに試すには [Playground](/shogi-board/playground/) から

## Option 1. Clone して使う

#### 必要なもの

Git・Node.js・Yarn  
↑ のインストール方法は [こちら](install_tools.md)

#### 手順

```sh
$ git clone https://github.com/murosan/shogi-board.git
$ cd shogi-board
$ yarn
$ yarn start
```
